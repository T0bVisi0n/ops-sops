# ops-sops

Operational system repository for SOPs, checklists, lead rules and internal business documentation.

## Purpose

This repository is the operational knowledge base of the business.

It stores the logic behind recurring tasks, decisions, workflows and standards so execution does not depend on memory or improvisation.

## Scope

- SOPs for recurring business processes
- checklists for execution and quality control
- lead qualification and cleanup rules
- operating principles and decision logs
- documentation for future scaling and delegation

## Goal

Turn scattered know-how into clear, repeatable systems that can be improved, reused and handed off without chaos.
